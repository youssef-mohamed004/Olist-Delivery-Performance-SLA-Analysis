# 🚚 Olist Delivery Performance & SLA Analysis

## 📌 Business Question
Are we meeting our delivery SLAs, and how does delivery performance impact customer satisfaction?

---

## 📊 Project Overview

This project analyzes the delivery performance of the Olist Brazilian E-Commerce Dataset to evaluate SLA compliance, identify the root causes of delivery delays, and measure their impact on customer satisfaction.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Microsoft PowerPoint
- Microsoft Word

---

## 📂 Dataset

- **Olist Brazilian E-Commerce Public Dataset (2017–2018)**
- Source: Kaggle

---

## 🔍 Key Findings

- ✅ 93.23% of orders were delivered on time.
- 🚚 90.2% of delivery delays originated during the carrier shipping stage.
- ⭐ Late deliveries reduced customer satisfaction by 47% (4.29 → 2.27).
- 📍 Northern states experienced the highest late delivery rates.
- 📦 Every 1% improvement in on-time delivery protects approximately 965 customers from a poor delivery experience.
